Exercício: Criando uma Tabela de Produtos

Objetivo: Criar uma tabela simples que exiba uma lista de produtos com suas respectivas informações.

Instruções:

  1. Crie uma tabela com as seguintes colunas:
    • Nome do Produto
    • Preço
    • Quantidade em Estoque
  2. Adicione pelo menos 3 produtos fictícios na tabela.
  3. Utilize corretamente as tags: <table>, <thead>, <tbody>, <tr>, <th> e <td>.
  4. (Opcional) Adicione um <tfoot> com uma linha mostrando o total de produtos listados.

Dica:

Você pode usar o atributo border="1" na tag <table> para facilitar a visualização da tabela.
