Exercício: Construindo um Calendário com Copilot

Objetivo: Criar um calendário com 3 meses utilizando tabelas HTML. Você pode utilizar o GitHub Copilot para ajudar na estruturação do código e preenchimento dos dados.

Instruções:

  1. Crie um novo arquivo chamado calendario.html no Visual Studio Code.
  2. Use <table> para criar a estrutura de cada mês do calendário. Devem ser 3 tabelas separadas: mês atual e os dois próximos.
  3. Adicione os dias da semana no <thead> de cada tabela (Dom, Seg, Ter, Qua, Qui, Sex, Sáb).
  4. Preencha os dias corretamente em cada mês. Use o GitHub Copilot para sugerir a estrutura de cada calendário (você pode começar com um comentário como: <!-- Calendário de Abril 2025 -->).
  5. Nos dias com feriados, adicione uma pequena descrição dentro da célula. Exemplo: 21 - Tiradentes.
  6. Adicione seus próprios agendamentos nos dias desejados (estudos, eventos, atividades pessoais).
  7. Você pode incluir links internos (ex: <a href="#agenda-estudos">) ou links externos (ex: <a href="https://example.com">) em algum dia com atividade especial.

Exemplo de comentários que você pode usar para acionar o Copilot:

  • <!-- Tabela do mês de Abril com os dias da semana -->
  • <!-- Marcar o dia 21 com feriado Tiradentes -->
  • <!-- Link para cronograma de estudos no dia 10 -->

Observações:

  • Esse calendário pode ser usado como seu cronograma pessoal de estudo ou organização mensal.
  • Capriche na clareza das informações e no uso dos links.
  • Você pode usar comentários para explicar o motivo de certas marcações, ajudando o Copilot a gerar sugestões melhores.
