Revisão de HTML

Este arquivo revisa as principais tags e seus usos no HTML.

Tags de Cabeçalho

As tags de cabeçalho vão de <h1> a <h6> e são usadas para títulos e subtítulos. <h1> é a mais importante, enquanto <h6> é a menos importante.

Parágrafos e Texto

A tag <p> define um parágrafo e é usada para agrupar blocos de texto.

  • <strong>: Destaca o texto com ênfase forte.
  • <em>: Destaca o texto com ênfase leve (itálico por padrão).
  • <mark>: Destaca trechos de texto com uma marcação colorida.
  • <small>: Exibe o texto em fonte menor.

Imagens

A tag <img> insere imagens na página. Alguns atributos importantes são:

  • src: Define o caminho ou URL da imagem.
  • alt: Fornece um texto alternativo se a imagem não puder ser exibida.
  • width e height: Controlam o tamanho exibido da imagem.

Outras Tags Semânticas

Além das tags básicas, HTML5 introduziu elementos semânticos que ajudam a organizar o conteúdo da página:

  • <header>: Cabeçalho ou introdução de uma seção ou página.
  • <nav>: Área de navegação com links.
  • <main>: Conteúdo principal da página.
  • <footer>: Rodapé da página.
  • <section>: Seção genérica de conteúdo.
  • <article>: Conteúdo independente e auto contido.
