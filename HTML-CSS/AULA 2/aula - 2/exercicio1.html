Exercício 1: Construindo seu Currículo com HTML

Objetivo: Construir seu currículo usando listas HTML.

Instruções:

  1. Formação Acadêmica:
    Crie uma lista com seus cursos, como:
    • Ensino Médio
    • Curso Técnico
    • Ensino Superior
  2. Experiência Profissional:
    Liste as experiências profissionais que você já teve, como:
    • Estágios
    • Trabalhos temporários
    • Empregos fixos
  3. Soft-Skills:
    Crie uma lista com suas principais soft-skills (habilidades comportamentais).
    Dica: Pense se essa lista deve ser ordenada ou não ordenada. Justifique sua escolha.
  4. Cursos e Aprimoramentos:
    Adicione cursos extras que você fez ou está fazendo. Para cada curso, crie uma sublista com os tópicos abordados.
    Exemplo:
    • Curso de HTML e CSS
      • Tags principais do HTML
      • Estilização com CSS
      • Responsividade
    • Curso de Lógica de Programação
      • Variáveis
      • Condicionais
      • Loops

Observação final:

Durante a construção do currículo, pense: Essas listas devem ser ordenadas, não ordenadas ou aninhadas?
Justifique sua escolha em um comentário dentro do código HTML.
