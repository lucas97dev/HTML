Como Instalar e Usar o GitHub Copilot

1. Criar uma Conta no GitHub

  1. Acesse o site: https://github.com
  2. Clique em Sign up no canto superior direito.
  3. Preencha seu e-mail, crie um nome de usuário e uma senha segura.
  4. Confirme o e-mail que será enviado para sua caixa de entrada.
  5. Complete a configuração inicial (não precisa escolher nenhum plano pago).

2. Instalar o Visual Studio Code (caso ainda não tenha)

  1. Acesse: https://code.visualstudio.com
  2. Baixe a versão para seu sistema operacional (Windows, macOS ou Linux).
  3. Instale normalmente como qualquer outro programa.

3. Instalar a Extensão GitHub Copilot

  1. Abra o Visual Studio Code.
  2. Clique no ícone de extensões na barra lateral esquerda ou pressione Ctrl + Shift + X.
  3. Pesquise por GitHub Copilot.
  4. Clique em Install na extensão oficial da Microsoft.

4. Configurar o GitHub Copilot

  1. Após a instalação, o VS Code vai pedir para você fazer login com sua conta do GitHub.
  2. Será aberta uma janela no navegador — clique em Authorize e siga as instruções.
  3. Depois do login, volte ao VS Code. O Copilot estará ativo.

5. Como Usar o GitHub Copilot

  1. Abra um novo arquivo (por exemplo, index.html ou script.js).
  2. Comece a digitar um comentário ou o início de uma função.
  3. O Copilot irá sugerir automaticamente trechos de código.
  4. Para aceitar uma sugestão, pressione Tab ou Enter.
  5. Se houver várias opções, use as setas ↑ ↓ para navegar entre elas.

Dica: O Copilot aprende com o contexto! Quanto mais descritivo seu código ou comentário, mais precisas serão as sugestões.
