Como Clonar um Repositório do GitHub

Objetivo: Aprender a copiar um repositório do GitHub para o seu computador local usando o comando git clone.

O que significa "clonar"?

Clonar é o processo de copiar um projeto que está no GitHub (ou outro servidor remoto) para o seu computador. Você poderá ver os arquivos, editar o projeto e até contribuir com ele.

1. Copie a URL do repositório

  1. Abra a página do repositório que você quer clonar no GitHub.
  2. Clique no botão verde "Code".
  3. Escolha a opção HTTPS (ou SSH, se souber usar) e clique em "Copy".
  4. Exemplo de URL: https://github.com/seu-usuario/seu-repositorio.git

2. Abra o terminal

Abra o terminal ou o Git Bash no local onde deseja salvar o repositório clonado (ex: sua pasta de projetos).

3. Use o comando git clone

git clone https://github.com/usuario/repositorio.git

Após rodar esse comando, o Git criará uma nova pasta com o nome do repositório e baixará todos os arquivos de código e histórico.

4. Acesse a pasta clonada

cd seu-repositorio

Agora você está dentro do projeto clonado e pode começar a trabalhar com ele localmente.

Dica:

  • Você pode clonar repositórios de outras pessoas (se forem públicos) para estudar, modificar ou contribuir.
  • Se você for colaborar, depois de clonar é importante criar seu próprio branch para não alterar o código da branch principal sem combinar.

Comando usado:

git clone [URL]

Exemplo prático:

git clone https://github.com/DiegoCampos1/html-css-ia.git
cd html-aula3
code .

code . abre o projeto no VS Code (se estiver instalado).
