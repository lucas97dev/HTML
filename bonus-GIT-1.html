Instalando o Git

Antes de começar a usar o Git, é necessário instalá-lo no seu computador.

Passos:

Instalação do Git, via site:

  1. Acesse: https://git-scm.com/
  2. O site já detecta seu sistema (Windows, Mac ou Linux). Clique em "Download".
  3. Execute o instalador e siga os passos padrão (pode manter as opções sugeridas).
  4. Após a instalação, abra o terminal e digite: 
    git --version
    Se o Git estiver instalado corretamente, você verá a versão atual.

Instalação do Git, via terminal:

  1. Abra o terminal.
  2. Para Windows, use o comando: 
    winget install --id Git.Git -e --source winget
  3. Para Mac, use o comando: 
    brew install git
  4. Para Linux, use o comando: 
    sudo apt-get install git
  5. Após a instalação, digite: 
    git --version
    Se o Git estiver instalado corretamente, você verá a versão atual.

Configuração Inicial:

Após a instalação, é importante configurar seu nome e email, que serão usados em seus commits.

Abra o terminal e digite:

git config --global user.name "Seu Nome"
git config --global user.email "Seu Email"

Substitua "Seu Nome" e "Seu Email" pelos seus dados reais.

Para verificar se as configurações foram salvas corretamente, use:

git config --global --list

Isso mostrará suas configurações atuais do Git.

Agora você está pronto para usar o Git!

Dica:

Instale também o GitHub Desktop se preferir uma interface gráfica para usar o Git: desktop.github.com

Com o vscode instalado, pode-se utilizar o git com o vscode
